title: Claude Code に Mastra ドキュメント MCPサーバを設定する
publishedAt: 2025-05-31

Claude Code（Anthropicが提供するコマンドラインツール）では、MCP（Model Context Protocol）サーバーを追加することで、外部のツールやドキュメントにアクセスできる。Mastra MCPサーバーを追加すると、IDE内から直接Mastraの全ナレッジベースにアクセスでき、以下のような情報を参照できる。

- ドキュメント、コード例、技術ブログ記事
- 機能のアナウンスやパッケージの変更履歴
- 最新のMCP（Mastra Cloud Platform）関連ニュース

具体的には、以下のような質問に答えてくれる。

**機能追加に関する質問**
- 「エージェントに評価機能を追加する方法は？」
- 「特定のタスク用のワークフローを作成するには？」
- 「サードパーティAPIにアクセスする新しいツールの作成方法は？」

**統合やデバッグに関する質問**
- AI SDKとの互換性について
- 特定の音声APIのサポート状況
- エージェントメモリーに関する最近の変更点
- ワーキングメモリーの仕組み

これによりMastraでのエージェント開発を効率化できる。

## 設定手順

Mastra MCPサーバーの追加は以下の手順で行える。

1. 以下のコマンドを実行して、Mastra MCPサーバーを追加する
```sh
claude mcp add mastra -- npx -y @mastra/mcp-docs-server@latest
```

2. MCPサーバーが正しく追加されたか確認する
```sh
$ claude mcp list
mastra: npx -y @mastra/mcp-docs-server@latest
```

## なぜMCPサーバーを使うのか？

- Claude Codeから離れることなくMastraのドキュメントにアクセスできる
- コンテキストに応じた適切なドキュメントを参照してくれる
- 最新のドキュメントを常に参照できる

## まとめ

MCPサーバーを使うことで、Claude Codeの機能を拡張し、外部ドキュメントへのアクセスが容易になる。特に開発中にドキュメントを頻繁に参照する場合、作業効率の向上が期待できる。
